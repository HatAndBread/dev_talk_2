CSS Pseudo Elements

Have you ever seen this syntax in CSS?

some_class::before {
  content: 'hello!'
} some_class::after {
  content: 'goodbye!'
}

Until recently every time I saw this syntax, I wondered what it did for a few seconds and then moved on.

Recently, however, I have found it to be super useful!

What is a pseudo-element? 🤔

A pseudo element simply mimics an HTML element without actually having to create a new element in the DOM.

Example without pseudo elements

MOUSE OVER ME! 🐭

Oh no! We have 2 useless divs! 😱

Example with pseudo elements

MOUSE OVER ME! 🐭

Yay! We don't have any useless divs! 🎉✨

Syntax

.some-class::before {
  content: '';
}

.some-class::after {
  content: '';
}

⚠️ You must add property "content" or it won't work!

⚠️ You can only use one before property and one after property
